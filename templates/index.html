SODA Tap is a temporal public data browser powered by the Socrata Open Data API (SODA). It was designed to be used as a library, but is also combined with a web server that exposes all public temporal data on the Socrata Platform via a web interface. Each temporal Socrata data resource is displayed with a chart or map, depending on the resource's data type (scalar or geospatial).

Socrata currently exposes ${totalSodaResources} data resources through their experimental search API. SODA Tap has identified ${totalTemporalResources} of them as temporal data streams, which can be browsed through this web application.
